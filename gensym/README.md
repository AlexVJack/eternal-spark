# GenSym — Fundamental Units of Sense / Фундаментальні одиниці сенсу

A generative symbol system for narrative fiction.
Генеративна символьна система для художньої прози.

---

## EN

### What is this

128 symbols — fundamental units of meaning that cover physics, mathematics, psychology, narrative, and human experience. Organized into 16 categories.

Not tarot. Not runes. No mystical baggage — just archetypes embedded in language and thought.

The logic: any combination of 4–7 symbols contains a story waiting to be interpreted.

### How to use

```bash
python3 gensym.py        # 4 symbols (default)
python3 gensym.py 7      # 7 symbols
python3 gensym.py 4 deep # 4 main symbols + 4 details for each
```

The script draws random symbols. You (or Claude) interpret the combination into narrative.

### How to interpret

There is no wrong interpretation. The symbols set a direction — not a script.

Look for:
- **Tension** between symbols (Chaos + Order, Birth + Death)
- **Sequence** — does one symbol lead to another?
- **The unexpected** — the combination that surprises you contains the story

### The 16 categories

Forces · Space & Movement · States · Actions · Relations · Archetypes · Elements · Psychology · Cognition · Temporal · Conflict · Nature & Cosmos · Body & Matter · Social · Abstract · Structural

## UA

### Що це

128 символів — фундаментальні одиниці сенсу що покривають фізику, математику, психологію, наратив і людський досвід. Організовані в 16 категорій.

Не Таро. Не руни. Без містичного забарвлення — тільки архетипи вбудовані в мову і мислення.

Логіка: будь-яка комбінація з 4–7 символів містить в собі історію яка чекає на інтерпретацію.

### Як використовувати

```bash
python3 gensym.py        # 4 символи (за замовчуванням)
python3 gensym.py 7      # 7 символів
python3 gensym.py 4 deep # 4 головних + 4 деталі для кожного
```

Скрипт витягує випадкові символи. Ти (або Claude) інтерпретуєш комбінацію в наратив.

### Як інтерпретувати

Неправильної інтерпретації немає. Символи задають напрямок — не сценарій.

Шукай:
- **Напругу** між символами (Хаос + Порядок, Народження + Смерть)
- **Послідовність** — чи веде один символ до іншого?
- **Несподіване** — комбінація що тебе здивувала містить в собі історію
