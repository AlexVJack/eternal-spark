# Naming System

Every name in Eternal Spark is born from a GenSym draw of 4 symbols.

## Structure

**Full name: [Given name] [Clan] [Region]**

- **Given name** — longer. Formed from the first two symbols: take opening syllables and merge.
- **Clan** — shorter. Formed from the third symbol: one syllable, slightly altered.
- **Region** — shortest. Formed from the fourth symbol: one syllable, compressed.

## How to read a name

A name carries meaning on two levels:
- Those who know the symbols read the character's nature directly in the name.
- Those who don't — simply hear the sound.

## Example — main character

**Draw:** Memory · Guilt · Disclosure · Synthesis

| Symbol | Element | Derivation |
|--------|---------|------------|
| Memory | Given name (part 1) | Me → Meg |
| Guilt | Given name (part 2) | Guil → gil |
| Disclosure | Clan | Dis → Dys |
| Synthesis | Region | Syn → Sin |

**Full name: Megil Dys Sin**
