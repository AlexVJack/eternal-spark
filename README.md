---

## How this book is written

Every scene begins with a draw.

**GenSym** is a generative symbol system — 128 fundamental units of meaning organized into 16 categories: Forces, Movement, States, Actions, Relations, Archetypes, Elements, Psychology, Cognition, Time, Conflict, Nature, Body, Social, Abstract, Structural.

A randomizer draws 4–7 symbols. Claude interprets the combination into narrative.

No mysticism. No tarot. Just archetypes — the ones already embedded in language, story, and physics.

The four symbols that started this book:

```
Catalyst · Intuition · Chaos · Revelation
```

See `concept.md` for what they revealed. See `gensym/` for the full symbol system.

## Як пишеться ця книга

Кожна сцена починається з розкладу.

**GenSym** — генеративна символьна система: 128 фундаментальних одиниць сенсу в 16 категоріях: Сили, Рух, Стани, Дії, Стосунки, Архетипи, Елементи, Психологія, Пізнання, Час, Конфлікт, Природа, Тіло, Соціальне, Абстрактне, Структурне.

Рандомайзер витягує 4–7 символів. Claude інтерпретує комбінацію в наратив.

Без містики. Без Таро. Тільки архетипи — ті що вже вбудовані в мову, оповідь і фізику.

Чотири символи що запустили цю книгу:

```
Каталіз · Інтуїція · Хаос · Одкровення
```

Дивись `concept.md` — що вони розкрили. Дивись `gensym/` — повна символьна система.
